Projects

Java Chess Game

Spring 2020

Overview

Challenged myself to utilize a new programming language, Java, to develop a functional chess game that can be run in the terminal.


Utilized inheritance and object oriented design in order to implement this program.

Details

The console outputs a visual representation of the board that will be updated after each move. To play enter the position notation of the piece you want to move and of where you want to move to. The program will ensure the move is legal before updating the board and switching whose turn it is. Note: there is no AI player, but you can play against yourself! :D


Start of chess game: At the beginning of the chess game the instuctions (boxed in red in the above left image), the position board (blue), and the starting position of the board (green) are all printed out


Taking a move: The above right image is a example of taking a move in this chess game. Note, the program rejects invalid positions and invalid moves (red underlines). After a valid move has been made a new, updated, game board is printed out (blue). Also, a list of captured enemy pieces (for the current player) are printed.

Take Aways

First personal Java programming project:

  • learned how to utilize object oriented design to organize & develop code
  • gained a deeper understanding of Java through the code development and debugging process
